GUIDED CORRECTION (Mapping)

Цель: превратить FAIL по гейтам в обязательный список уточнений для клиента.

Связка: Gate → Что не хватает → Какие вопросы INTAKE уточнить

Gate0 (Envelope)
- Нет статуса/режима/версий
- Уточнить: Q1, Q2, Q3

Gate1 (Case + Owner)
- Нет ясного владельца или зоны ответственности
- Уточнить: Q2

Gate2 (System Boundary)
- Не определены границы системы
- Уточнить: Q5

Gate3 (Interaction)
- Нет ключевых акторов/связей
- Уточнить: Q5 + добавить конкретные элементы/потоки

Gate4 (Contradiction)
- Нет явного trade‑off
- Уточнить: Q6

Gate5 (IKR)
- Нет целевого состояния
- Уточнить: Q4

Gate6 (Hard Barrier)
- Нет жёсткого ограничения
- Уточнить: Q3

Gate7 (Resources)
- Не указаны ресурсы
- Уточнить: Q7

Gate8 (Form Selection)
- Нет логики выбора формы
- Уточнить: Q6 + Q7

Gate9 (Solution Set)
- Нет вариантов решений
- Уточнить: Q5 + Q7

Gate10 (Mechanism Specs)
- Нет механизма/шагов
- Уточнить: Q5 + Q8

Gate11 (Secondary Check)
- Нет проверки вторичного противоречия
- Уточнить: Q6

Gate12 (Decision Record)
- Не зафиксировано решение
- Уточнить: Q1 + Decision Lock блок

Gate13 (Implementation Map)
- Нет плана и ответственных
- Уточнить: Q2 + Q8

Gate14 (Rule Update)
- Нет правила/границ
- Уточнить: Q3 + Q7
